## The Proactive Agent Mirage

Everyone wants an AI agent that texts the patient after 3 days to remind them of their medication, or proactively notifies them that their results are ready. At the LLM level, it's easy: you inject a deferred task and let the model draft an empathetic message.

But when you build on top of Meta, reality hits you hard.

## What Nobody Tells You About the WhatsApp API

If more than 24 hours have passed since the last message the user sent you, the customer service window closes. If you try to send a free-text message generated by your LLM through the official WhatsApp API, Meta blocks it and returns a silent error. Your agent can't speak freely.

## The Real Data in Our Asynq Queue

While building our new B2C personal agent in Go, we realized that all our long-term reminder jobs were failing. The Asynq task queue processed the job correctly, the LLM generated the message, but the Meta API systematically rejected it.

## What We Built: The Template Workaround

To bypass this regulatory restriction by Meta, proactive reminders MUST be sent using a pre-approved "Utility Template". But how do you maintain the AI's personalization using a static template?

Our Go architecture now intercepts the elapsed time. If the 24-hour window has expired, the orchestrator takes the text generated by the LLM Engine and injects it into a generic template with a single dynamic variable (`{{1}}`).

```go
// If 24h passed, we inject the LLM output into the approved template
if time.Since(lastUserMessage) > 24*time.Hour {
    err := whatsappClient.SendTemplateMessage(userPhone, "recordatorio_agente", []string{
        llmGeneratedMessage, // Injected into {{1}}
    })
    if err != nil {
        log.Errorf("Failed to send utility template: %v", err)
    }
} else {
    // If inside the window, send free text
    whatsappClient.SendTextMessage(userPhone, llmGeneratedMessage)
}
```

## The Gotcha: The Business Model Breaks

The technical workaround is simple, but the business impact is brutal. WhatsApp Utility templates cost money per send (it varies by country, but in Chile, it's a fixed cost per 24-hour conversation initiated by the business).

If you sell your agent as a flat $10/month subscription, and the agent "proactively" decides to text the patient 15 times a month outside the 24h window, Meta will eat your entire profit margin.

## Zoom Out: Orchestrating Against the Rules

Building health tech over WhatsApp isn't just about dealing with AI hallucinations or optimizing prompts; it's a constant battle against Meta's commercial limits and anti-spam policies. The real challenge of modern Healthtech is designing a technical architecture that protects your finances without ruining the user experience.
